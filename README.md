### Usage

Install the package

```js
npm install --save react-flexbox-z
```

Import the module in the place you want to use:
```js
import { FlexItem, FlexGroup } from "react-flexbox-z";

```

#### Snippet
```js
<FlexGroup<"div"> tagName="div">
</FlexGroup>

// default div
<FlexGroup>
</FlexGroup>
```
